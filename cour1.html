Chapitre 1 : Notion de Programme en Python

Comprendre les fondements théoriques d'un programme informatique et le fonctionnement de l'interpréteur Python.

1. Qu'est-ce qu'un programme informatique ?

Un programme informatique est une séquence structurée d'instructions rédigées dans un langage formel, conçue pour être exécutée par un ordinateur afin de résoudre un problème algorithmique ou d'accomplir une tâche spécifique.

En Python, le code source est conservé dans un fichier texte portant l'extension .py. Ces instructions sont ensuite lues et exécutées séquentiellement, ligne par ligne, par l'interpréteur Python.

2. Premiers exemples de code

Exemple 1 : Calcul d'énergie cinétique

Un premier script appliqué à la physique pour calculer l'énergie cinétique ($E_c = \frac{1}{2}m v^2$) d'un solide en mouvement :

# Calcul de l'énergie cinétique (en Joules)
masse = 1.5 # masse en kg
vitesse = 10.0 # vitesse en m/s

energie_cinetique = 0.5 * masse * (vitesse ** 2)
print("L'énergie cinétique est de :", energie_cinetique, "J")
Résultat d'exécution en console :
L'énergie cinétique est de : 75.0 J

Exemple 2 : Calcul d'une moyenne pondérée

Un exemple montrant l'utilisation des variables et des opérateurs pour calculer une moyenne avec coefficients :

# Calcul d'une note finale avec coefficients
note_ds = 14.5
note_tp = 16.0

coef_ds = 3
coef_tp = 1

moyenne = (note_ds * coef_ds + note_tp * coef_tp) / (coef_ds + coef_tp)
print("La moyenne finale est :", moyenne)
Résultat d'exécution en console :
La moyenne finale est : 14.875

3. Les spécificités du langage Python

Un langage de programmation sert d'interface de communication entre le concepteur humain et la machine. Python se distingue par les caractéristiques suivantes :

  • Langage interprété : Le code source est traduit et exécuté au fur et à mesure sans phase de compilation préalable.
  • Typage dynamique : La détermination du type des variables est gérée automatiquement lors de l'exécution.
  • Syntaxe épurée et rigoureuse : La structure des blocs de code repose obligatoirement sur l'indentation, garantissant une lisibilité maximale.
  • Portabilité (Multi-plateforme) : Compatibilité native avec les systèmes Windows, macOS et Linux.
  • Bibliothèque standard riche : Accès direct à de nombreux modules scientifiques, mathématiques et de gestion de données.

4. Cycle de développement d'un script Python

  1. Édition : Rédaction du code source dans un environnement de développement (IDE comme VS Code, PyCharm, Thonny, etc.).
  2. Interprétation : Lecture et exécution séquentielle des instructions par le moteur Python.
  3. Affichage : Restitution des résultats dans la console de sortie.
  4. Débogage : Analyse des erreurs de syntaxe ou de logique pour optimisation du code.
