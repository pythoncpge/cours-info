Chapitre 2 : Les Types de Données de Base

Vue d'ensemble des structures et types élémentaires manipulés par l'interpréteur Python.

1. Les Nombres int float complex

Python distingue trois types numériques principaux pour le calcul mathématique :

  • Entiers (int) : Nombres sans partie décimale, de précision illimitée.
  • Flottants (float) : Représentation des réels selon la norme IEEE 754.
  • Complexes (complex) : Stockent la partie réelle et imaginaire (notée avec j).
a = 10 # int
pi = 3.14159 # float
z = 2 + 3j # complex

2. Les Chaînes de Caractères str

Séquence immutables de caractères textuels encadrés par des guillemets simples ou doubles.

nom = "Alice"
message = 'Bonjour CPGE'

3. Les Booléens bool

Types logiques binaires prenant pour valeur True ou False. Indispensables pour le contrôle de flux.

est_valide = True
condition = (10 > 5) # Évalue à True

4. Les Séquences list tuple range

Collections d'éléments ordonnés :

  • list : Modifiable (mutable).
  • tuple : Fixe / Non modifiable (immutable).
  • range : Séquence d'entiers générée à la volée.
notes = [12, 15, 18] # Liste
vecteur = (1.0, 0.0, 2.5) # Tuple
indices = range(0, 10) # Range

5. Ensembles & Dictionnaires set dict

Structures de données non séquentielles :

  • Ensemble (set) : Collection non ordonnée d'éléments uniques.
  • Dictionnaire (dict) : Structure d'association par clés/valeurs.
fruits = {"pomme", "orange"}
profil = {"nom": "Alice", "note": 16}
