💡 Les Variables en Python

Définition, règles de nommage, affectation et types de données

🧩 1. Définition

Une variable est un espace nommé dans la mémoire de l'ordinateur permettant de stocker une valeur afin de la réutiliser ultérieurement dans votre programme.

Analogie : On peut imaginer une variable comme une boîte étiquetée. La boîte contient la valeur, et l'étiquette (le nom) permet d'y accéder. 

# Exemples d'initialisation de variables
note_examen = 18.5
nom_etudiant = "Youssef"

Ici, note_examen contient la valeur numérique 18.5 et nom_etudiant contient la chaîne de caractères "Youssef".

🧠 2. Règles de nommage

  • Doit commencer par une lettre (a-z, A-Z) ou un underscore (_).
  • Ne doit jamais commencer par un chiffre.
  • Ne peut contenir ni espaces ni caractères spéciaux (sauf _).
  • Sensible à la casse : Score, score et SCORE sont 3 variables distinctes.

✅ Exemples valides

x = 10
moyenne_general = 15.25
_valeur_temp = 30

❌ Exemples invalides

1er_nombre = 5   # Débute par un chiffre
nom complet = "A" # Contient un espace
prix€ = 100       # Caractère interdit (€)

⚙️ 3. Affectation de valeurs

L'affectation se fait grâce à l'opérateur d'égalité = :

a = 42
pi = 3.14159
message = "Bienvenue en CPGE !"

# Affichage dans la console
print(a)
print("Valeur de pi :", pi)

🔄 Affectations multiples & Échange

# 1. Affectation multiple simultanée
x, y, z = 10, 20, 30

# 2. Échange direct de deux variables (Swap)
a, b = 5, 9
a, b = b, a  # Maintenant a vaut 9 et b vaut 5

# 3. Même valeur pour plusieurs variables
p1 = p2 = p3 = 0

📦 4. Types de données fondamentaux

Python est un langage à typage dynamique : le type de la variable est déterminé automatiquement selon la valeur attribuée.

nom = "Sara"       # str (String / Chaîne de caractères)
age = 19           # int (Integer / Entier)
note = 16.75       # float (Nombre à virgule flottante)
est_inscrit = True # bool (Boolean / Vrai ou Faux)
