Chapitre 4 : Les Entrées et Sorties (I/O) en Python

Interagir avec l'utilisateur : récupérer des données de saisie et structurer l'affichage des résultats.

1. Lecture des données : La fonction input()

L'interaction utilisateur repose sur la récupération d'informations externes. La fonction prédéfinie input() met le programme en attente d'une saisie au clavier jusqu'à ce que la touche Entrée soit pressée.

print("Initialisation du programme...")
utilisateur = input("Saisissez votre pseudo : ")
Affichage dans le terminal :
Initialisation du programme...
Saisissez votre pseudo : Alex

2. Gestion des types et conversion des données

La fonction input() renvoie toujours une valeur de type chaîne de caractères (str). Pour effectuer des opérations arithmétiques, il est indispensable de procéder à un cast (conversion explicite de type).

A. Problématique du typage par défaut

age = input("Quel est votre âge ? ")
# Provoque une erreur TypeError si on tente : print(age + 1)

B. Application du cast (Conversion)

# Conversion directe en entier (int)
valeur_age = int(input("Entrez votre âge : "))
print("Dans un an, vous aurez :", valeur_age + 1, "ans.")

# Exemples pour d'autres types réels ou booléens
note = float(input("Note du DS (ex: 15.5) : "))
statut = bool(int(input("Actif (1) ou Inactif (0) : ")))

3. Affichage des résultats : La fonction print()

La sortie standard s'effectue via print(). On peut lui passer un ou plusieurs arguments séparés par des virgules.

nom = "Sophie"
score = 95
print("Candidat :", nom, "| Note obtenue :", score)
Résultat d'exécution :
Candidat : Sophie | Note obtenue : 95

4. Paramètres avancés de print()

Il est possible de modifier le comportement d'affichage grâce aux arguments optionnels sep (séparateur d'éléments) et end (caractère de fin de ligne).

# Personnalisation du séparateur avec sep
print("2026", "07", "22", sep="/")
# Sortie : 2026/07/22

# Empêcher le retour à la ligne automatique avec end
print("Chargement", end="...")
print("Terminé !")
# Sortie : Chargement...Terminé !

5. Exemple d'application globale

prenom = input("Entrez votre prénom : ")
annee_naissance = int(input("Année de naissance : "))
age_calcule = 2026 - annee_naissance
print("Ravi de vous rencontrer", prenom, "!", "Vous avez environ", age_calcule, "ans.")

6. Fiche Mémorisation (Synthèse)

📥 Entrée Standard

Lecture des données saisies par l'utilisateur au clavier.

variable = input("Message")

🔄 Transtypage (Cast)

Convertit la chaîne lue en type numérique requis.

int(), float(), str()

📤 Sortie Standard

Publication et évaluation des expressions dans la console.

print(valeur1, valeur2)

⚙️ Options d'Affichage

Formatage fin pour les séparateurs et fins de lignes.

sep="...", end="..."
